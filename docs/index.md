# EasyMenu

EasyMenu is a lightweight, header-only C++ library for building simple, console-based interactive menus. It provides a small, focused API to create menus, add and manage items, and display a user-friendly navigation UI on multiple platforms.

This documentation covers installation, quick start examples, core concepts, and a complete reference for the public API exposed by the `include/` folder.

- Library: EasyMenu
- Location: `include/` (public headers)
